Shops List on Map

This module allows you to create a list of shops to display on an interactive map.

Also this documentation is available in other languages as English | Russian | Spanish

Adaptive

Automatic geolocation at the address of the shop. Accuracy of geolocation of 30 meters

Informative

All contact data on the shop card. Shop work schedule by days of the week. Embedding photos of the shop

Flexible

Work beautifully on desktop and mobile devices. Usage of own snippet with Yandex map engine or Partner Google Map Snippet to show the map

Selective

Assign shop properties for dynamic or permanent filtering

Main features

Shop geolocation

Display of the shop on the map with full information

Route to the shop

Get directions to the selected shop

Shop filtering

Assign special properties to filter shop list showing

Search

Display nearest shops to the desired location

Images

Add images for more shop's presentation to display...

...on the shop card or on the personal shop's webpage.

Mobile-friendly

Ready to use on devices with small screen size
