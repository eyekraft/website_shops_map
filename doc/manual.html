The order of creating store

1. Open menu Inventory/Warehouse management/Warehouses.

2. Create new warehouse:

  • check "Shop" checkbox to assign warehouse as public shop
  • specify the name of the shop
  • specify the short name of the shop
  • specify city where shop is located in the field "City"
  • specify address of the shop in the field "Street"
  • specify the name of the mall in the field "Mall" if any
  • specify the name of the subway station if any
  • select properties for shop (properties must be pre-filled - see below)
  • enter opening days in format of your own (ex. «mo-fr», «Saturday»), or select option filled earlier
  • enter opening hours, or select option filled earlier
  • load images for the shop to display on shop page, if any

3. Save the form. Now shop is available to show on the snippet map

Shop properties

Properties are used to filter shops on the webpage. They can be selected by user from dropdown list or be preinstalled on snippet load permanently with no selection ability.

1. Open menu Inventory/Warehouse Management/Shop properties

2. Create new property:

  • specify name of the property (to see in the dropdown selection on the webpage)
  • specufy 'Sequence' if necessary (properties with a lower number are shown in the selection list earlier)
  • check 'Default' to apply this property automatically on snippet load
  • specify URL-address to apply automatic property on specified page only, ex. «/page/shops»*

3. Save the form. The property is available in the property list of the shop creation form

* If "URL" field is filled the automatic filter works on specified page only. Otherwise, filter applies on every shop map snippet of the website.
