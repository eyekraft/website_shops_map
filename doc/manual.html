This instructions allows you to configure the Odoo backend, all the necessary registers to be able to show the list of shops.

Shop API Users

To register Odoo database as shop list source you need to generate local API-key.

1. Open menu Inventory/Warehouse Management/Shop Management/API Settings/Shop API Users

  • Enter URL to current database and get you API-key on pushing saving button.
  • Use this URL and generated API-key on map Shop List Snippet source setup.

Yandex Maps API

1. Open menu Settings/General Settings/Yandex Maps View

2. Set API key and map localization, visit the page about get an API Key for the "JavaScript API y HTTP Geocoder" Map service.

Fill all of the required data and save it.

Multi-Warehouses

You can manage several warehouses. You need to have Manager Access Rights for Inventory module to work with shops.

1. Open menu Inventory/Warehouse Management/Shop Management/Multi-Warehouses

Fill all of the required data and save it.

Shop List Config

Create a Shop List Config because in the snippet configuration dialog you can choose next options of your shop list.

1. Open menu Inventory/Warehouse Management/Shop Management/Shop List Config

  • specify "Shop List Url", defines the URL address for Shop List.
  • specify "Shop List Paramters", defines the URL address for Shop List.
    • specify "Colour", defines colour of shop card border. Cause snippet allows to show lists from several sources it may be need to distinguish between them.
    • specify "Label", defines short text mark placed before shop name. It has the same purpose as colour.
    • specify "Info", defines the type of information to display in the shop card caption. It's distance from current user's location to the shop location by default.
    • specify "Shop tags", defines shops with what tags will be shown on the map of the snippet. Cause this module supports Partner Google Map Snippet it can use its tagged Partners with geolocation completed. Partners available for Partner Google Map Snippet are imported as shops on this module install and are able to show by corresponding tag. The module's own tag is 'shop' and it is used by default.
  • specify "Widget ID", defines the Widget ID for Shop List.

Periods of Time

Create a Periods of Time entering opening hours for the shop.

1. Open menu Inventory/Warehouse Management/Shop Management/Periods of Time

  • specify "Since" defines the hour from in which the shop works, in set hours in numbers.
  • specify "To" defines the hour until in which the shop works, in set hours in numbers.

Shop Work Days

1. Open menu Inventory/Warehouse Management/Shop Management/Shop Work Days

Create a opening days in format of your own (ex. «Mon-Fri», «Saturday») for the shop.

Fill all of the required data and save it.

Shop properties

Properties are used to filter shops on the webpage. They can be selected by user from drop-down list or be pre-installed on snippet load permanently with no selection ability.

1. Open menu Inventory/Warehouse Management/Shop Management/Shop properties

2. Create new property:

  • specify "Name" of the property (to see in the drop-down selection on the webpage)
  • specify "Sequence" if necessary (properties with a lower number are shown in the selection list earlier)
  • check "Default" to apply this property automatically on "Shop List Snippet" load
  • specify "URL-address" to apply automatic property on specified page only, ex. «/page/shops»*

3. Save the form. The property is available in the property list of the shop creation form

* If "URL" field is filled the automatic filter works on specified page only. Otherwise, filter applies on every "Shop List Snippet" of the website.

The order of creating shop

You can to create Warehouse as Public Shop.

1. Open menu Inventory/Warehouse Management/Warehouses

2. Create a new Warehouse:

  • Mark it as Public Shop, checkbox to assign warehouse as public shop.
    • specify the name of the shop (used to display on the shop map with the "Shop List Snippet")
    • specify a short name of the shop (arbitrary alphanumeric designation)
    • specify the name of city where shop is located in the field "City"
    • specify address of the shop as the street, building number and building name in the field "Street" field
    • specify the name of the mall in the field "Mall", if necessary
    • specify the name of the subway station in the "Metro" field, if necessary
    • select the services available in the shop in the "Properties" field (services must be pre-configured - see below)
    • add a shop hours entry
    • enter opening days in format of your own (ex. «Mon-Fri», «Saturday»), or select previously entered days from the list
    • enter opening hours and enter the data in the window that appears, or select previously entered hours from the list
    • upload photos in the "Images" tab for the shop to display on shop page, if necessary

3. Save the form. Now shop is available to show on the "Shop List Snippet" map

Remember: You can select the services available in the shop in the "Properties" field (services must be pre-configured - see previous section).

Remember: You can enter opening days in format of your own (ex. «Mon-Fri», «Saturday»), or select previously entered days from the list.

Also you can enter opening hours and enter the data in the window that appears, or select previously entered hours from the list.

Shop Coordinate Adjustment

Sometimes a situation may arise when the address of the shop does not accurately reflect the location of the shop on the map. In this case:

1. Go to the «Yandex maps» resource (map.yandex.ru)

2. Enter your shop address in the search bar

3. If the shop is not exactly indicated on the map, find the correct location visually and click on it

4. A pop-up help window will show the geo-coordinates of the point

5. Copy these coordinates into the fields "Latitude" and "Longitude" on the form with the data of the shop

6. Save the form (Save)
