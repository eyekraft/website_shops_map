Esta instrucciones le permite configurar el el backend de Odoo, todos los registros necesarios para poder mostrar el listado de tiendas.

Usuarios de API de la tienda

Para registrar la base de datos de Odoo como fuente de lista de tienda, debe generar una clave API local.

1. Abrir el menú Inventario/Gestión de almacén/Gestión de la tienda/Configuración de API/Usuarios de API de la tienda

  • Ingrese la URL a la base de datos actual y obtenga su clave API al presionar el botón Guardar.
  • Use esta URL y la clave API generada en la configuración de la fuente del fragmento HTML de Shop List Snippet para mapa.

Yandex Maps API

1. Abrir el menú Ajustes/Opciones Generales/Vista de mapas de Yandex

2. Debe establecer API Key y localización de mapas, visite la página acerca de obtener una API Key para el servicio de mapas "JavaScript API y HTTP Geocoder".

Llena todos los datos requeridos y guárdelo.

Multialmacén

Puede gestionar varios almacenes. Debe tener derechos de acceso de administrador para el módulo de Inventario para trabajar con tiendas.

1. Abrir el menú Inventario/Gestión de almacén/Gestión de la tienda/Multialmacén

Llena todos los datos requeridos y guárdelo.

Configuración de lista de tiendas

Crear una configuración de lista de tiendas porque en el cuadro de diálogo de configuración de fragmentos puede elegir las siguientes opciones de su lista de compras.

1. Abrir el menú Inventario/Gestión de almacén/Gestión de la tienda/Configuración de lista de tiendas

  • "URL de lista de la tienda", define la dirección URL para lista de la tienda.
  • "Parámetros de lista de Tienda", define los parámetros para Lista de Tienda, como:
    • "Color", define el color del borde de la tarjeta de la tienda. Por que el fragmento HTML le permite mostrar listas de varias fuentes, es posible que sea necesario distinguirlas.
    • "Etiqueta", define una marca de texto corta colocada antes del nombre de la tienda. Tiene el mismo propósito que el color.
    • "Información", define el tipo de información que se mostrará en el título de la tarjeta de la tienda. Es la distancia desde la ubicación del usuario actual hasta la ubicación de la tienda predeterminada.
    • "Etiquetas de tienda", define las tiendas con qué etiquetas se mostrarán en el mapa del fragmento. Debido a que este módulo es compatible con Partner Google Map Snippet, puede usar sus socios etiquetados con geolocalización completa. Los socios disponibles para Partner Google Map Snippet se importan como tiendas en la instalación de este módulo y se pueden mostrar con la etiqueta correspondiente. La etiqueta propia del módulo es 'shop' y se usa predeterminada.
  • "ID de Widget", define el ID de Widget para la Lista de Tienda.

Períodos de tiempo

Crear un Período de Tiempo ingresando el horario de apertura de la tienda.

1. Abrir el menú Inventario/Gestión de almacén/Gestión de la tienda/Períodos de tiempo

  • "Desde", define la hora a partir de la cual trabaja la tienda, en horas establecidas en números.
  • "A", define la hora hasta la que trabaja la tienda, en horas establecidas en números.

Días de trabajo de la tienda

Cree un día de apertura en su propio formato (por ejemplo, «lunes a viernes», «sábado») para la tienda.

Propiedades de Tienda

Las propiedades se utilizan para filtrar tiendas en la página web. Pueden ser seleccionados por el usuario de la lista desplegable o estar preinstalados en la carga de fragmentos de forma permanente sin capacidad de selección.

1. Abrir el menú Inventario/Gestión de almacén/Almacenes/Gestión de la tienda/Propiedades de tienda

2. Crear nueva propiedad:

  • especifique el "Nombre" de la propiedad (para ver en la selección desplegable en la página web)
  • especifique la "Secuencia" si es necesario (las propiedades con un número más bajo se muestran en la lista de selección anterior)
  • marque como "Predeterminado" para aplicar esta propiedad automáticamente en la carga del fragmento HTML de "Shop List Snippet"
  • especifique la dirección "URL" para aplicar la propiedad automática solo en la página especificada, ej. «/page/shops»*

3. Guarde el formulario. La propiedad está disponible en la lista de propiedades del formulario de la creación de tiendas.

* Si el campo "URL" está lleno, el filtro automático funciona solo en la página especificada. De lo contrario, el filtro se aplica a cada fragmento HTML de "Shop List Snippet" de tienda del sitio web.

El orden de creación de la tienda

You can to create Warehouse as Public Shop.

1. Abrir el menú Inventario/Gestión de almacén/Almacenes

2. Crear un nuevo Almacén:

  • Marque la casilla de verificación Tienda pública, para asignar el almacén como tienda pública.
    • especifique el nombre de la tienda (usado para mostrar en el mapa de la tienda con el fragmento HTML de "Shop List Snippet")
    • especifique un nombre corto de la tienda (designación alfanumérica arbitraria)
    • especifique el nombre de la ciudad donde se encuentra la tienda en el campo "Ciudad"
    • especifique la dirección de la tienda como: calle, número y nombre del edificio en el campo "Calle"
    • especifique el nombre del centro comercial en el campo "Centro comercial", si corresponde
    • especifique el nombre de la estación de metro en el campo "Estación de metro", si corresponde
    • seleccione los servicios disponibles en la tienda en el campo "Propiedades" (los servicios deben estar preconfigurados - ver a continuación)
    • añadir una entrada de horario de la tienda
    • ingrese los días de apertura en su propio formato (p. ej., «Lunes a Viernes», «Sábado»), o seleccione las horas ingresadas previamente de la lista
    • ingrese el horario de apertura e ingrese los datos en la ventana que aparece, o seleccione el horario ingresado previamente de la lista
    • cargue fotos en la pestaña "Imágenes" para que la tienda las muestre en la página de la tienda, si corresponde

3. Guarde el formulario. Ahora la tienda está disponible para mostrar en el fragmento HTML de "Shop List Snippet"

Recuerde: seleccione los servicios disponibles en la tienda en el campo "Propiedades" (los servicios deben estar preconfigurados - ver sección anterior).

Recuerde: ingrese los días de apertura en su propio formato (p. ej., «Lunes a Viernes», «Sábado»), o seleccione las horas ingresadas previamente de la lista.

También usted puede ingrese el horario de apertura e ingrese los datos en la ventana que aparece, o seleccione el horario ingresado previamente de la lista.

Ajuste de coordenadas de la tienda

A veces puede surgir una situación en la que la dirección de la tienda no refleje con precisión la ubicación de la tienda en el mapa. En este caso:

1. Ir al recurso «Yandex maps» (map.yandex.ru)

2. Introduce la dirección de tu tienda en la barra de búsqueda

3. Si la tienda no está exactamente indicada en el mapa, busque la ubicación correcta visualmente y haga clic en ella

4. Una ventana emergente de ayuda mostrará las coordenadas geográficas del punto

5. Copia estas coordenadas en los campos "Latitud" y "Longitud" del formulario con los datos de la tienda

6. Guarde el formulario
