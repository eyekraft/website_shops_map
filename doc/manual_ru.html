Пользователи API магазинов

Для того, чтобы использовать текущую базу в качестве источника списка магазинов, необходимо сформировать локальный ключ API.

1. Откройте меню Инвентаризация/Управление складом/Управление магазином/API Settings/Пользователи API магазинов

  • Введите URL текущего сайта, и в процессе сохранения записи будет сформирован ключ API.
  • Введите этот URL и ключ API во время настройки источника сниппета карты.

Yandex Maps API

Shop List Config

Create a Shop List Config because in the snippet configuration dialog you can choose next options of your shop list.

1. Откройте меню Инвентаризация/Управление складом/Управление магазином/Shop List Config

  • specify "Shop List Url", defines the URL address for Shop List.
  • specify "Shop List Paramters", defines the URL address for Shop List.
    • specify "Colour", defines colour of shop card border. Cause snippet allows to show lists from several sources it may be need to distinguish between them.
    • specify "Label", defines short text mark placed before shop name. It has the same purpose as colour.
    • specify "Info", defines the type of information to display in the shop card caption. It's distance from current user's location to the shop location by default.
    • specify "Shop tags", defines shops with what tags will be shown on the map of the snippet. Cause this module supports Partner Google Map Snippet it can use its tagged Partners with geolocation completed. Partners available for Partner Google Map Snippet are imported as shops on this module install and are able to show by corresponding tag. The module's own tag is 'shop' and it is used by default.
  • specify "Widget ID", defines the Widget ID for Shop List.

Свойства магазинов

Свойства используются в качестве фильтров, сортируя магазины по услугам. Они могут выбираться поcетителем сайта из выпадающего меню, а также устанавливаться автоматически при загрузке страницы.

1. Откройте меню Инвентаризация/Управление складом/Управление магазином/Свойства магазина

2. Создайте новое свойство (Create):

  • укажите название свойства (используется в списке выбора)
  • при необходимости укажите последовательность (Sequence) (свойства с меньшим номером показываются в списке раньше)
  • при необходимости активируйте опцию «По-умолчанию» (Default), чтобы автоматически применять это свойство при загрузке страницы
  • укажите в поле «URL» адрес страницы для автоматического фильтра, например «/page/shops»*

3. Сохраните форму (Save). Свойство станет доступно в поле «Свойства» при создании магазинов.

* При заполненном URL фильтр работает только на указанной станице. Если это поле остается пустым, фильтр применяется ко всем страницам со списком магазинов.

Порядок создания магазина

Вы можете создать склад как общедоступный магазин.

1. Откройте меню Инвентаризация/Управление складом/Склады

2. Создайте новый склад склад (Create):

  • активируйте опцию Магазин, флажок, чтобы назначить склад как публичный магазин.
    • укажите название (используется для отображения на карте магазинов)
    • укажите короткое название (произвольное буквенно-цифровое обозначение)
    • укажите город в поле «Город»
    • укажите улицу, номер строения и название строения в поле «Улица»
    • укажите название торгового заедения а поле «Торговый центр»
    • при необходимости укажите название станции метро в поле «Метро»
    • выберите услуги, доступные в магазине в поле «Свойства» (услуги должны быть предварительно сформированы - см. ниже)
    • добавьте запись часов работы магазина
    • выберите создание дней работы магазина и введите данные в произвольном формате (например, «пн-пт», «суббота»), либо выберите из списка ранее введенные дни
    • выберите создание часов работы и введите данные в появившемся окне, либо выберите из списка ранее введенные часы
    • при необходимости загрузите фотографии во вкладке «Изображения

3. Сохраните форму (Save). Магазин станет доступен для показа на карте

Помните: Вы можете выбрать доступные в магазине услуги в поле "Свойства" (услуги должны быть предварительно настроены - см. предыдущий раздел).

Помните: Вы можете выбрать доступные в магазине услуги в поле "Свойства" (услуги должны быть предварительно настроены - см. предыдущий раздел).

Также вы можете ввести часы работы и ввести данные в появившемся окне или выбрать ранее введенные часы из списка.

Корректировка координат магазина

Иногда может возникнуть ситуация, когда адрес магазина не точно отражает метоположение магазина на карте. В этом случае:

1. Пройдите на ресурс «Яндекс-карты» (map.yandex.ru)

2. Введите в строке поиска адрес вашего магазина

3. Если магазин не точно указан на карте, найдите правильное место визуально и щелкните по нему мышью

4. Всплывающее справочное окно покажет гео-координаты пункта

5. Скопируйте эти координаты в поля Широта (Latitude) и Долгота (Longitude) на форме с данными магазина

6. Сохраните форму (Save)
